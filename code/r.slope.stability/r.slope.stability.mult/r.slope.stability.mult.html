DESCRIPTION

r.slope.stability - The slope stability model is designed for physically-based slope stability analysis for single slopes or larger areas (up to hundreds of square kilometres).

For details, please refer to the link below.

REFERENCE

Please visit the r.slope.stability web site for further information and a detailed manual: https://www.landslidemodels.org/r.slope.stability
